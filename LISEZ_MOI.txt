=== STREAM SCREEN ===

Application de streaming d'écran optimisée pour Raspberry Pi.

LANCEMENT :
Double-cliquez sur "start.bat" pour lancer l'application.
(Cela vérifie les mises à jour et lance l'interface graphique sans console).

FONCTIONNALITÉS :
1. Interface "Stream Screen" moderne.
2. Bouton unique Lancer/Stopper.
3. Onglet "Raspberry Pi" pour lancer le récepteur à distance via SSH.
   - Le mot de passe est sauvegardé.
   - Le chemin du fichier est configurable (par défaut: Desktop/stream_receiver.py).
   - Le stream se lance automatiquement côté PC si nécessaire.

FICHIERS :
- StreamScreen.pyw : L'application principale.
- stream_receiver.py : Le script à mettre sur le Raspberry Pi.
- stream_config.json : Vos préférences (Sauvegardées automatiquement).
- start.bat : Le lanceur recommandé.

DÉPANNAGE :
Si le stream ne se lance pas sur le Pi :
- Vérifiez l'IP (ex: 192.168.x.x).
- Vérifiez que le fichier est bien au chemin indiqué (Desktop/stream_receiver.py).
- Consultez "sender_debug.log" en cas de problème technique.
